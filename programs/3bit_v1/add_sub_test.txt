# Add/Sub 3-bit v1 test program
# filename: add_sub_test
# tests all integer adds and subs using loops
# 3-bit v1 ISA:
# 000 HALT   - Stop execution
# 001 MOVL   - Move literal: litA -> [BC]
# 010 ADD    - Add: [A] + [B] -> [C]
# 011 SUB    - Subtract: [A] - [B] -> [C]
# 100 CMP    - Compare: flags <- [0:A] vs [B:C]
# 101 JEQ    - Jump if equal: if flag then goto ABC
# 110 JGT    - Jump if greater: if flag then goto ABC
# 111 NOP    - No operation

001 000 000 000 # 0 - move literal 0 into RAM addr 0 (A)
001 001 000 001 # 1 - move literal 1 into RAM addr 1 (B)
001 001 000 010 # 2 - move literal 1 into RAM addr 2 (1 for incrementing)
001 000 000 011 # 3 - move literal 0 into RAM addr 3 (0 for jumping when B overflows)
010 000 001 100 # 4 - ADD #0 + #1 -> #4
111 100 001 100 # 5 - MOVOUT: copy result at #4 (page0) -> same addr on page1
011 000 001 101 # 6 - SUB #0 - #1 -> #5
111 101 001 101 # 7 - MOVOUT: copy result at #5 (page0) -> same addr on page1
010 001 010 001 # 8 - ADD #1 + #2 -> #1 (increment B)
100 001 011 000 # 9 - CMP #1 to #3 (zero)
110 000 000 100 # 10 - JMP to #4 if B>0 (inner loop)
010 000 010 000 # 11 - ADD #0 + #2 -> #0 (increment A)
100 000 011 000 # 12 - CMP #0 to #3 (zero)
110 000 000 100 # 13 - JMP to #4 if A>0 (outer loop)
000 000 000 000 # 14 - Halt execution
