# CMP Page test
# tests writing and comparing to data on other pages
# 3-bit v1 ISA:
# 000 HALT   - Stop execution
# 001 MOVL   - Move literal: litA -> [BC]
# 010 ADD    - Add: [A] + [B] -> [C]
# 011 SUB    - Subtract: [A] - [B] -> [C]
# 100 CMP    - Compare: sets flags based on [A] vs [B]
# 101 JEQ    - Jump if equal: if flag then goto ABC
# 110 JGT    - Jump if greater: if flag then goto ABC
# 111 NOP    - No operation


001 001 010 000 # 0 - write 1 to page 2 addr 0
001 011 001 001 # 1 - write 3 to page 1 addr 1
001 010 000 001 # 2 - write 2 to page 0 addr 1
001 010 100 010 # 3 - write 2 to page 4 addr 2

100 001 100 010 # 4 - compare &0-1 (2) to &4-2 (2)
101 000 000 111 # 5 - if equal, jmp to line 7 (should pass)
001 001 000 111 # 6 - if jeq fails, write 1 to &0-7 (should skip)

100 001 100 010 # 7 - compare &0-1 (2) to &4-2 (2)
110 000 001 010 # 8 - if greater, jmp to line 10 (should fail)
001 010 000 111 # 9 - if jgt fails, write 2 to &0-7 (should occur)

100 001 001 001 # 10 - compare &0-1 (2) to &1-1 (3)
110 000 001 101 # 11 - if greater, jmp to line 13 (should fail)
001 011 000 111 # 12 - if jgt fails, write 3 to &0-7 (should occur)

100 001 010 000 # 13 - compare &0-1 (2) to &2-0 (1)
110 000 010 000 # 14 - if greater, jmp to line 16 (should pass)
001 100 000 111 # 15 - if jgt fails, write 4 to &0-7 (should skip)

001 111 000 111 # 16 - finalize by writing 7 to &0-7
